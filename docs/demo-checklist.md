# SalesGuard - Demo checklist Sprint 1

## Preparacion

1. Ejecutar `npm install`
2. Ejecutar `npm run dev`
3. Confirmar que el servidor arranque en `http://localhost:3000`

## Flujo de demo

1. Abrir `GET /api`
2. Mostrar `GET /api/health`
3. Abrir `GET /docs`
4. Explicar la arquitectura inicial del backend
5. Explicar el diseno inicial de base de datos
